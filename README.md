# Bank-Simulation-Java
A simple object-oriented program that simulates a bank managing multiple accounts.

### What it does
- Create bank accounts with a unique number and starting balance
- Deposit and withdraw money from any account
- Get the balance of any account
- Calculate the total balance of all accounts
- Count how many accounts have at least a certain balance
- Find an account by its number
- Find the account with the largest balance
- Convenience methods to add/deposit/withdraw/get balance using account number

### How to run
1. Make sure you have Java installed
2. In the project folder open terminal or command prompt
3. Compile all files:
4. Run the demo program java BankAccountTester
